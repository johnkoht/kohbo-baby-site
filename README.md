# Middleman Base

This repo is a base template for creating new Middleman projects. It's fairly opinionated.

## Setup

**Clone the repo**

```bash
$ git clone https://github.com/kohactive/middleman_base
```


**Run the setup script**

```bash
$ bin/setup
```

## Run the project

To run Middleman, simple call:

```bash
$ middleman
```
